実数値の式(Champions modの検証済み仕様。個体値31固定・レベル非依存):
```
Spe = trunc(trunc((種族値 + SP + 20) * 110, 16bit) / 100)   # 性格↑
    = 種族値 + SP + 20                                     # 性格無補正
    = trunc(trunc((種族値 + SP + 20) * 90, 16bit) / 100)   # 性格↓
```
