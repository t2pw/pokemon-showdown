# M-B Single Battle Party Notes

## Purpose

This file summarizes the current Pokemon party discussion for external analysis.

Format: Regulation M-B, single battle, 3v3 team preview.

The original core is built around Corviknight and Primarina using pivot moves to create safer entry opportunities for Mega Blaziken.

## Current Core

### Blaziken @ Blazikenite

- Form target: Mega Blaziken
- Ability after Mega: Speed Boost
- Nature: unknown
- EVs: H32 / D32 / B2, as provided
- Moves:
  - Flare Blitz
  - Close Combat
  - Thunder Punch
  - Protect

Role:

- Main late-game cleaner / physical attacker.
- The team tries to use U-turn and Flip Turn to bring Blaziken in without taking unnecessary damage.
- Protect is used to safely gain Speed Boost.

Notes:

- Current set is more of an immediate attacker than a dedicated setup sweeper.
- Swords Dance is a possible option if the team needs Blaziken itself to break defensive teams, but Thunder Punch currently helps into Water/Flying targets.

### Corviknight @ Leftovers

- Ability: unknown
- Nature: unknown
- EVs: H32 / B32 / C2, as provided
- Moves:
  - Body Press
  - U-turn
  - Iron Defense
  - Roost

Role:

- Physical wall and pivot.
- Uses U-turn to generate Blaziken entry.
- Can become a win condition with Iron Defense + Body Press in some matchups.

Notes:

- Helps stabilize physical matchups.
- May overlap somewhat with Mega Scizor if Mega Scizor is considered as a second Mega option.

### Primarina @ Sitrus Berry

- Ability: unknown
- Nature: unknown
- EVs: unknown
- Moves:
  - Sparkling Aria
  - Moonblast
  - Flip Turn
  - Misty Terrain

Role:

- Bulky special attacker / pivot.
- Flip Turn supports the core game plan by bringing Blaziken in safely.
- Misty Terrain was chosen specifically as a strong answer to Hippowdon, especially to block Yawn while keeping momentum with Flip Turn.

Notes:

- Encore was discussed as an alternative, but Misty Terrain should remain a serious option because it directly addresses Hippowdon, which is otherwise heavy for the basic selection.

## Main Concerns

- The last three slots are not decided.
- The team may lack convincing answers to:
  - Screens
  - Stall / defensive loops
  - Hippowdon if Primarina is not selected or loses momentum
  - Electric pressure into Corviknight + Primarina
- There is interest in running a second Mega Pokemon as an alternative Mega selection.

## Discussed Direction

Recommended six from the current discussion:

1. Mega Blaziken
2. Corviknight
3. Primarina
4. Garchomp
5. Archaludon
6. Mega Gengar

Reasoning:

- Garchomp gives an Electric immunity, Stealth Rock, Focus Sash lead value, and possible Brick Break for screens.
- Archaludon checks many common Dragon/Fairy-adjacent matchups and can provide phazing with Dragon Tail.
- Mega Gengar gives a second Mega option that does not overlap heavily with Mega Blaziken. It improves defensive matchups, stall pressure, and some screen/setup matchups through trapping, Taunt, Encore, Perish Song, or Nasty Plot.

## Candidate Additions

### Garchomp @ Focus Sash

- Ability: Rough Skin preferred
- Nature: Jolly preferred
- EVs: A32 / S32 / remaining EVs flexible
- Moves:
  - Earthquake
  - Dragon Claw / Scale Shot
  - Stealth Rock
  - Brick Break / Rock Tomb

Role:

- Electric immunity.
- Stealth Rock setter.
- Lead or emergency stopgap.
- Brick Break is specifically for screens; Rock Tomb is more generally useful for speed control.

### Archaludon @ Chople Berry or other non-conflicting item

- Ability: Stamina or Sturdy, to be decided
- Nature: Calm or bulky nature, to be decided
- EVs: H32 / D32 / remaining EVs flexible
- Moves:
  - Draco Meteor
  - Flash Cannon
  - Dragon Tail
  - Stealth Rock / Body Press

Role:

- Bulky Steel/Dragon slot.
- Checks or soft-checks many high-usage threats.
- Dragon Tail helps against setup and screen turns.

Notes:

- Thunder Wave was considered, but Misty Terrain on Primarina can conflict with status-based plans. Dragon Tail and Stealth Rock may fit better if Misty Terrain stays.

### Gengar @ Gengarite

- Form target: Mega Gengar
- Nature: Timid preferred
- EVs: C32 / S32 / remaining EVs flexible
- Moves:
  - Shadow Ball
  - Sludge Bomb
  - Perish Song / Nasty Plot
  - Taunt / Encore / Destiny Bond

Role:

- Second Mega option.
- Anti-stall / anti-defensive-cycle tool.
- Can punish passive targets, trap key Pokemon, or force trades.
- Provides a distinct game plan from Mega Blaziken.

Notes:

- Mega Gengar is preferred over Mega Gyarados, Mega Scizor, and Mega Floette in this structure because it overlaps less with the current core.
- If using Mega Gengar, Blaziken is not always selected. The team should be treated as a two-Mega preview team with matchup-dependent Mega choice.

## Alternative Second Mega Options Considered

### Mega Gyarados

Pros:

- Strong setup threat.
- Can be a powerful alternative win condition.

Cons:

- Adds another Water-type next to Primarina.
- Can worsen Electric pressure.
- Role may be less clean than Mega Gengar in this specific core.

### Mega Scizor

Pros:

- Strong priority and pivot potential.
- Good with a slower, positioning-heavy style.

Cons:

- Overlaps with Corviknight as a Steel pivot / physically bulky piece.
- Does not solve stall and defensive loops as directly as Mega Gengar.

### Mega Floette

Pros:

- Strong special Fairy Mega.
- High environment relevance.

Cons:

- Overlaps with Primarina's Fairy/special attacking role.
- Does not directly support the Blaziken pivot plan as cleanly.

## Possible Selection Patterns

### Standard Blaziken Plan

- Primarina + Corviknight or Garchomp + Mega Blaziken
- Goal: pivot with Flip Turn / U-turn, then land Blaziken safely and clean late.

### Screen Matchup

- Garchomp + Mega Gengar + Mega Blaziken or Archaludon
- Goal: use Brick Break, Taunt/Encore, Dragon Tail, or trapping to prevent free setup behind screens.

### Stall / Defensive Cycle

- Mega Gengar + Primarina + Garchomp or Archaludon
- Goal: use trapping, Taunt, Perish Song, Nasty Plot, or hazard pressure instead of relying only on Blaziken.

### Hippowdon Matchup

- Primarina should usually be strongly considered.
- Misty Terrain blocks Yawn and allows Primarina to maintain momentum with Flip Turn.
- Garchomp or Archaludon can add Stealth Rock / Dragon Tail pressure.

## Key Questions For Analysis

1. Is Mega Gengar the best second Mega for this structure, or is another Mega more coherent?
2. Should Blaziken remain four attacks + Protect, or should Swords Dance be used over Thunder Punch or Close Combat?
3. Is Misty Terrain Primarina worth keeping over Encore given the stated Hippowdon concern?
4. Should Garchomp or Archaludon be the primary Stealth Rock setter?
5. Does the proposed six have enough immediate damage into bulky Water/Fairy cores?
6. Are there matchup holes against top M-B threats such as Garchomp, Archaludon, Basculegion, Mega Floette, Corviknight, and Primarina?
